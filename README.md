# Zioxs' Portfolio Website

Welcome to the source code repository for my professional portfolio website, showcasing my work as a Minecraft Mod Developer. 

🌐 **Live Website:** [https://zioxs.github.io](https://zioxs.github.io)

## About This Repository

This project is a modern, responsive, and dark-themed portfolio built using **Next.js**. It serves as the primary landing page for my Minecraft modding and basic 3D modeling services. 

Key features of this website include:
- **Project Showcase:** Highlights custom Forge and Fabric mods with tailored descriptions and links.
- **Service Listings:** Detailed breakdown of my mod development and 3D modeling offerings.
- **Client Testimonials:** Real reviews from content creators and clients I have worked with.
- **Localization:** Full bilingual support (English and Indonesian) seamlessly built into the UI.

## Tech Stack
- **Framework:** Next.js (Static Export)
- **Styling:** CSS Modules
- **Icons:** Lucide React
- **Animations:** Framer Motion
- **Deployment:** GitHub Pages (Automated via GitHub Actions)
